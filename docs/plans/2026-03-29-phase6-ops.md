# Phase 6: AI Ops Layer

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add the AI observability layer: structured JSON logging, `/ops/status` diagnostics endpoint, runtime stats collection with DB persistence, error tracking, and a built-in HTML dashboard. This is what differentiates Brace from every other framework.

**Architecture:** A `Stats` singleton collects per-request metrics (latency, status codes, query counts) in lock-free atomic counters. A ring buffer stores per-minute aggregates. Errors are deduplicated and stored in an `ops_errors` DB table. Stats flush to `ops_stats` DB table once per minute. The `/ops/status` endpoint returns everything as JSON. The dashboard is a single embedded HTML page that polls this endpoint.

**Tech Stack:** java.util.concurrent.atomic (counters), SLF4J + Logback (structured logging), JUnit 5

```java
package com.larvalabs.brace;

import java.time.Instant;
import java.util.LinkedHashMap;
import java.util.Map;

public class Log {

    // Structured event logging — outputs JSON to stdout
    public static void event(String event, Map<String, Object> data) {
        var entry = new LinkedHashMap<String, Object>();
        entry.put("ts", Instant.now().toString());
        entry.put("level", "INFO");
        entry.put("event", event);
        entry.putAll(data);
        System.out.println(toJson(entry));
    }

    // Internal: log a request completion
    static void request(String method, String path, int status, long durationUs,
                        int queryCount, long queryUs) {
        var entry = new LinkedHashMap<String, Object>();
        entry.put("ts", Instant.now().toString());
        entry.put("level", status >= 500 ? "ERROR" : "INFO");
        entry.put("event", "http.request");
        entry.put("method", method);
        entry.put("path", path);
        entry.put("status", status);
        entry.put("durationMs", durationUs / 1000.0);
        entry.put("queries", queryCount);
        entry.put("queryMs", queryUs / 1000.0);
        System.out.println(toJson(entry));
    }

    // Internal: log an error
    static void error(String method, String path, Throwable error) {
        var entry = new LinkedHashMap<String, Object>();
        entry.put("ts", Instant.now().toString());
        entry.put("level", "ERROR");
        entry.put("event", "http.error");
        entry.put("method", method);
        entry.put("path", path);
        entry.put("error", error.getClass().getSimpleName());
        entry.put("message", error.getMessage());
        System.out.println(toJson(entry));
    }

    private static String toJson(Map<String, Object> map) {
        // Use Jackson for reliable JSON — Json.mapper() is already available
        try {
            return Json.mapper().writeValueAsString(map);
        } catch (Exception e) {
            return map.toString();
        }
    }
}
```

- [ ] **Step 1: Implement Stats**

Stats collects per-request metrics using atomic counters (lock-free, zero contention). Maintains a ring buffer of per-minute aggregates.

```java
package com.larvalabs.brace;

import java.time.Instant;
import java.util.*;
import java.util.concurrent.ConcurrentHashMap;
import java.util.concurrent.atomic.AtomicLong;
import java.util.concurrent.atomic.LongAdder;

public class Stats {

    // Current minute accumulators
    private final LongAdder requestCount = new LongAdder();
    private final LongAdder errorCount = new LongAdder();
    private final LongAdder totalLatencyUs = new LongAdder();
    private final LongAdder totalQueryCount = new LongAdder();
    private final LongAdder totalQueryUs = new LongAdder();
    private final AtomicLong maxLatencyUs = new AtomicLong(0);
    private final ConcurrentHashMap<Integer, LongAdder> statusCodes = new ConcurrentHashMap<>();
    private final ConcurrentHashMap<String, RouteStats> routeStats = new ConcurrentHashMap<>();

    // Ring buffer: per-minute snapshots (last 60 minutes)
    private final MinuteSnapshot[] minuteBuffer = new MinuteSnapshot[60];
    private int minuteIndex = 0;

    // Error tracking: recent unique errors
    private final List<ErrorRecord> recentErrors = Collections.synchronizedList(new ArrayList<>());
    private static final int MAX_RECENT_ERRORS = 50;

    private final Instant startedAt = Instant.now();

    // Record a completed request
    public void recordRequest(String method, String path, int status, long latencyUs,
                              int queryCount, long queryUs) {
        requestCount.increment();
        if (status >= 500) errorCount.increment();
        totalLatencyUs.add(latencyUs);
        totalQueryCount.add(queryCount);
        totalQueryUs.add(queryUs);
        maxLatencyUs.accumulateAndGet(latencyUs, Math::max);
        statusCodes.computeIfAbsent(status, k -> new LongAdder()).increment();

        var routeKey = method + " " + path;
        routeStats.computeIfAbsent(routeKey, k -> new RouteStats()).record(latencyUs);
    }

    // Record an error
    public void recordError(String type, String message, String route, String stackTrace,
                            String requestDetail, String queriesBefore) {
        // Deduplicate by type + route
        synchronized (recentErrors) {
            for (var err : recentErrors) {
                if (err.type.equals(type) && err.route.equals(route)) {
                    err.count++;
                    err.lastSeen = Instant.now();
                    err.lastStackTrace = stackTrace;
                    err.lastRequestDetail = requestDetail;
                    err.lastQueriesBefore = queriesBefore;
                    return;
                }
            }
            if (recentErrors.size() >= MAX_RECENT_ERRORS) {
                recentErrors.remove(0);
            }
            recentErrors.add(new ErrorRecord(type, message, route, stackTrace,
                requestDetail, queriesBefore));
        }
    }

    // Snapshot current minute and rotate
    public MinuteSnapshot rotateMinute() {
        var snapshot = new MinuteSnapshot(
            Instant.now(),
            requestCount.sumThenReset(),
            errorCount.sumThenReset(),
            totalLatencyUs.sumThenReset(),
            maxLatencyUs.getAndSet(0),
            totalQueryCount.sumThenReset(),
            totalQueryUs.sumThenReset()
        );
        minuteBuffer[minuteIndex % 60] = snapshot;
        minuteIndex++;
        return snapshot;
    }

    // Getters for ops endpoint
    public Instant startedAt() { return startedAt; }
    public long totalRequests() { return requestCount.sum(); }
    public long totalErrors() { return errorCount.sum(); }
    public Map<Integer, Long> statusCodeCounts() {
        var result = new LinkedHashMap<Integer, Long>();
        statusCodes.forEach((k, v) -> result.put(k, v.sum()));
        return result;
    }
    public List<MinuteSnapshot> minuteSnapshots() {
        var list = new ArrayList<MinuteSnapshot>();
        for (var s : minuteBuffer) {
            if (s != null) list.add(s);
        }
        return list;
    }
    public List<ErrorRecord> recentErrors() { return List.copyOf(recentErrors); }
    public Map<String, RouteStats> routeStats() { return Map.copyOf(routeStats); }

    // Data classes
    public record MinuteSnapshot(Instant ts, long requests, long errors,
        long totalLatencyUs, long maxLatencyUs, long queries, long queryUs) {
        public double avgLatencyMs() { return requests > 0 ? (totalLatencyUs / (double) requests) / 1000.0 : 0; }
    }

    public static class RouteStats {
        private final LongAdder count = new LongAdder();
        private final LongAdder totalLatencyUs = new LongAdder();
        public void record(long latencyUs) { count.increment(); totalLatencyUs.add(latencyUs); }
        public long count() { return count.sum(); }
        public double avgLatencyMs() { return count.sum() > 0 ? (totalLatencyUs.sum() / (double) count.sum()) / 1000.0 : 0; }
    }

    public static class ErrorRecord {
        public final String type;
        public final String message;
        public final String route;
        public String lastStackTrace;
        public String lastRequestDetail;
        public String lastQueriesBefore;
        public Instant firstSeen;
        public Instant lastSeen;
        public int count;

        ErrorRecord(String type, String message, String route, String stackTrace,
                    String requestDetail, String queriesBefore) {
            this.type = type;
            this.message = message;
            this.route = route;
            this.lastStackTrace = stackTrace;
            this.lastRequestDetail = requestDetail;
            this.lastQueriesBefore = queriesBefore;
            this.firstSeen = Instant.now();
            this.lastSeen = Instant.now();
            this.count = 1;
        }
    }
}
```

- [ ] **Step 1: Create OpsHandler**

OpsHandler provides these endpoints (all require `?key=SECRET`):
- `GET /ops/status` — full JSON diagnostics
- `GET /ops/routes` — route table as JSON

The `/ops/status` response includes:
- app: name, mode, uptime, startedAt, javaVersion, framework version
- http: totalRequests, errorRate, statusCodes, slowestRoutes (top 5 by avg latency)
- memory: heapUsed, heapMax, gcPauses (from ManagementFactory)
- errors: recent unique errors with count, stack trace, request detail
- jobs: scheduled job statuses (from JobScheduler) + durable job stats (if DB available)
- websockets: placeholder (Phase not implemented yet)
- mailer: sent count, captured emails in dev mode
- timeseries: minute snapshots from ring buffer

OpsHandler takes Stats, JobScheduler, Mailer, Router, DatabaseFactory, and the ops secret as constructor params.

Auth check: if `?key=` param doesn't match ops secret, return 401.
